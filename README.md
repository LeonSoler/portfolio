### Prerequisites
- A computer (optional, but recommended).
- **Node.js**: New enough to not complain, old enough to be stable.
- **Yarn**: Because `npm` is for mortals.

### Installation
1.  Clone this repo (and star it, or my self-esteem will crash):
    ```bash
    git clone https://github.com/leonsoler/portfolio.git
    cd portfolio
    ```

2.  Install dependencies (this is the part where you go get a coffee):
    ```bash
    yarn install
    ```

3.  Run the development server:
    ```bash
    yarn dev
    ```
    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. If it doesn't load, try turning it off and on again.
